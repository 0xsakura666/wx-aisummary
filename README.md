# wx-aisummary
通过分析微信群聊天记录获得总结
1、下载旧版微信：3.9.12.56
2、因为直接登陆会跳请更新新版本，所以需要使用低版本绕过修复软件（建议选择“新方案”）
3、将手机聊天记录迁移至电脑
4、安装memotrace，先点击“获取信息”，再点击“解析数据”，每次如果有新的聊天记录更新则需要点击“增量解析”
5、选择对应聊天社群，点击右上角聊天记录，选择导出TXT,右上角时间选择自定义，建议选择今天的日期，下面消息类型只选择“文本”，导出获得txt文件
6、使用wx-aisummary工具，选择聊天记录txt文件，点击分析，30秒左右即可出总结
